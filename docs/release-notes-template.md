# DSA Sheet Desktop vX.Y.Z

## Highlights

- Adaptive next-problem guidance built into the desktop app
- Monaco-based in-app Java editor
- Local Run and Submit workflow
- Topic progression across Arrays, Bit Manipulation, Linked List, Stack, Queue, Recursion, Binary Search, Trees, Graphs, and Dynamic Programming

## Included Platforms

- macOS
- Windows
- Linux

## Release Channel

- macOS: stable
- Windows: preview
- Linux: preview

## Important Requirement

DSA Sheet currently runs Java solutions locally on your machine.

Please install **JDK 17 or newer** to use Run and Submit.

If Java is missing, the app will show a setup banner and guide you.

## What’s Inside

- Current Task → Solve → Feedback → Next Task workflow
- Topic-aware learning progression
- Local workspace management
- Real compile/run-based checking for problems with test cases
- Concept-aware feedback and recommendations

## Download

Choose the installer for your platform from the release assets:

- macOS: `.dmg`
- Windows: `.exe`
- Linux: `.AppImage`

## Windows Preview Note

The Windows build may show an "Unknown publisher" or trust warning if it is unsigned.

That is expected for the free-first release path.

If Windows warns on install:

1. choose **More info**
2. choose **Run anyway**

We plan to move to a signed Windows release in a later version.
